สำหรับ Solution 1 คือการสร้าง Node ขึ้นมาก่อน node ทั้งหมด แล้วให้ weight เป็น 0 แล้วทำ Bellman ไปที่ node ที่สร้างใหม่

แต่ใน Solution ที่ 2 จะกลับด้าน edges ทั้งหมด แล้ว ก็ทำ Bellman จาก node v แล้วมา loop ดูว่าที่ node เริ่มอันไหนน้อยสุด

## Note ในข้อนี้ จริงๆแล้วเค้าบอกไม่มี Negative Cycle เพราะงั้นไม่ต้องเช็คก็ได้ และที่สำคัญ (ทำ Bellman แค่ n-1 รอบก็พอ) (ปกติรอบที่ n จะทำเพื่อเช็ค cycle บางทีอาจจะ กระทบทุก edges ซ้ำ 1 ครั้งก็ได้ แต่ในโค้ดจะ loop n รอบเลย แล้วเช็คว่าค่ามีการอัพเดตไหม)
